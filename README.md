# Dynamical Network Biomarker 动态网络标志物

本项目包含RNA-seq(转录组测序)、Chip-seq(染色质免疫共沉淀)、Proteomics(蛋白质组学)，这三大类生物数据的动态网络标志物以及相关数据差异表达分析。
